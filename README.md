EasyBook is a web-based appointment scheduling application that allows clients to book available appointment time slots with a single service provider. The system automatically generates availability, prevents double-booking, and provides an administrative interface for managing schedules and appointments.

This project was developed as a senior capstone project to demonstrate full-stack web development, backend scheduling logic, and database integration.

## Key Features

- View available appointment dates and time slots  
- Book appointments through a simple web interface  
- Prevent double-booking through conflict detection  
- Admin dashboard for managing availability and appointments  
- Persistent data storage using a relational database

- HTML

**Backend**
- Python

**Database**
- SQLite (development)  
- PostgreSQL (production)

---

## Project Scope

EasyBook is designed for a single service provider operating in one time zone with fixed appointment durations. Advanced features such as payments, multi-provider scheduling, and external calendar synchronization are outside the scope of this project.
